- About Bally's Interactive

This is the start of my career as a full-fledged front-end developer! We use JIRA for task management, familiar with Scrum & Kanban frameworks, and a fan of documentation or Mural/Miro whiteboards!

I have handled more than 8 brands to maintain and improve site performance. I mostly work on HTML and CSS styling, with some minor JavaScript tasks. Many of the tasks I've taken on involve revamping and implementing new features.

I've learned a lot, especially in terms of local environment setup, transitioning from React to the Lit framework, using SCSS/SASS, CI/CD Pipelines and understanding Git workflow.
